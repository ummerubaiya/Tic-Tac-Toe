Tic-Tac-ToeThis is an advanced Tic-Tac-Toe game built with HTML, CSS, and JavaScript. It offers multiple grid sizes, difficulty levels, and game modes, providing fun and challenge for all players

# Features:

Grid sizes: 3x3 (classic) and 4x4 (harder challenge)
Game modes: Single Player vs AI, Player vs Player
AI difficulty: Easy (random moves), Medium (basic strategy), Hard (Minimax algorithm)
Interactive UI: Highlights winning combinations and detects draws
Restart button: Start a new game anytime
Responsive design: Works on desktop and mobile screens
Leaderboard System: Automatically records player scores and displays top results

# How to Play

Open home.html in your browser.
Click Start game button to begin 
Choose game mode: Single Player vs AI or Player vs Player.
Choose difficulty level easy(3x3) , medium (3x3) or hard (4x4).

If playing vs AI, select difficulty level.

Players take turns placing X or O.

First to get three (or four for 4x4) in a row wins.

If all cells are filled and no winner, the game ends in a draw.

Click Restart to start a new game.
